=== CONTENT BUILDER ===

Author: Al Nemec
Requires at least: 3.4
Version: 1.0.0

== Description ==

Content Builder is a Word press Plugin which lets you create multiple page layouts.


== Requirements ==

- jQuery at least 1.7.2, jQuery UI at least 1.8.2
- PHP 5 with GD library;
- Permission to `/wp-content/uploads/`

== Installation ==

1. Upload the `content-builder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Got to `Options`, `Content Builder` to set it up
