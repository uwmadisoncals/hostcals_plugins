Weaver II Maintenance Tools

== Description ==

This plugin provides some important, but not frequently used, tools for maintaining your Weaver II site.

1. Clean all Weaver II settings from the WordPress database.

2. Upgrade Weaver 2.2.x - upgrade and convert your settings from Weaver 2.2.x and Weaver Plus 2.2.x.

== Installation ==

It is easiest to use the Plugin Add Plugin page, but you can do it manually, too:

1. Download the plugin archive and expand it
2. Upload the weaver-cleaner.php file to your wp-content/plugins/weaver-cleaner directory
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for Weaver Cleaner.
